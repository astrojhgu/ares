Problem Types
=============
There are several pre-defined problem types one can access via the parameter
``ptype``.


``ptype`` :math:`\leq 20`
--------------------------
These are all 1-D radiative transfer problems.

``ptype`` :math:`\geq 100`
--------------------------
These are all uniform background or reionization problems.
